#  Deep Learning Experiments

This folder contains implementations and experiments created while following **Andrej Karpathy's "Neural Networks: Zero to Hero"** lecture series.

The goal of this section is to understand **how deep learning works internally**, rather than just using high-level libraries.

---

## Learning Source

These implementations are inspired by the lecture series:

**Neural Networks: Zero to Hero – Andrej Karpathy**

The series explains how modern deep learning systems work by building them from scratch step by step.

Topics covered include:

* Backpropagation
* Automatic differentiation
* Neural networks
* Language models
* Training pipelines

---

## Contents

### micrograd

A tiny automatic differentiation engine that implements backpropagation.

Concepts explored:

* computational graphs
* gradient propagation
* chain rule
* scalar autograd system

---

### makemore

A character-level language model that learns to generate text.

Concepts explored:

* embeddings
* neural network language models
* probability distributions
* training loops

---

### neural network experiments

Simple neural network implementations used to understand:

* forward propagation
* loss functions
* gradient descent
* parameter updates

---

## Goals

This folder is intended for:

* learning deep learning fundamentals
* understanding how frameworks like PyTorch work internally
* experimenting with neural network architectures

---

## Running the Experiments

Install dependencies:

```bash
pip install torch numpy
```

Run a script:

```bash
python micrograd.py
```

or

```bash
python makemore.py
```
